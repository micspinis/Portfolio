Text Similarity API

Este proyecto fue iniciado este año (2020) como un ejercicio para implementar de manera productiva un servicio usando un modelo pre-entrenado, en este caso de NLP e implementarlo en mi universidad para que los profesores pudieran detectar tareas copiadas.

Con esta API el usuario puede ingresar dos archivos de texto y conocer su similitud. La Api también cuenta con validadción de usuario y un sistema de tokens que sirven como metodo de "pago", aun que el objetivo no es el cobro, me parecio una buena practica para limitar su uso.

Este proyecto se encuentra en su primer versio y aun no esta implementado en ningun servidor. Todo esta dentro de un conteiner de docker para su protabilidad, se implementa un base de datos no relacional y un modelo pre-entrenado de NLP de uso libre de la gente de spaCy.

El usuario deberá registrarse, al registrarse obtendra un numero de tokens, si el nombre de usuario esta disponible puede continuar. Ahora debe validar la entrada con su nombre de usuario y contraseña para poder hacer uso de la Api, una vez validado puede ingresar los dos textos y recibira un valor cuantitativo de 0 a 1, donde estos valores representan el porcentaje de similitud en los textos.

Esta API se puede considerar con un servicio back-end parte de un proyecto mas grande.

Tecnologias:
  • - Python
  • - Flask
  • - Mongo DB
  • - Docker
  • - PostMan
