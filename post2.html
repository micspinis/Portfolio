Crypto Coin

Este proyecto fue iniciado este año con el objetivo de poner en práctica mis nuevas habilidades usando React.

La aplicación cuanta con una interfaz agradable donde el usuario puede seleccionar el tipo de moneda y la criptomoneda con la que desea realizar la cotización.

La aplicacion cuenta con validación de campos.

Al presionar el boton de calcular, se lanza un spinner mientras se realiza la consulta a la Api. El resultado arroja el precio de la criptomoneda seleccionada en términos de la modena de cambio.

Además el usuario recibe información extra como lo es el precio mas alto y mas bajo del día, variación en las últimas 24 hrs y el tiempo de la última actualización.

Tecnologias:
  • - JavaScript
  • - React
  • - Node.js
